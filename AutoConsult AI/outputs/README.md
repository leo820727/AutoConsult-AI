# outputs/ — 診斷報告輸出目錄

此資料夾存放 AutoConsult AI 自動生成的診斷報告。

## 輸出文件類型

1. **`.pdf`**: 專業版診斷報告（含圖表與格式樣式，建議直接給客戶）。
2. **`.md`**: Markdown 原始報告（適用於進一步編輯或貼入其他文件）。
3. **`.json`**: AI 原始分析數據（適用於二次開發或數據串接）。

## 預設命名規則
```text
[YYYY-MM-DD]_[公司名稱]_report.[副檔名]
```

## 注意事項
- 請勿手動修改此目錄下的 JSON 檔案，以免影響報告生成的完整度。
- 若部署至 GitHub，建議將此目錄加入 `.gitignore`。
